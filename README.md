# 项目简介

## 核心功能
- 基于YOLOv8的水稻病害检测系统，支持识别3类病害：Bacteria Leaf Blight、Brown Spot、Leaf smut
- 提供6715张标注水稻病害数据集
- 支持多种输入方式：单张图片、图片文件夹、视频文件、本地摄像头
- 实时显示目标位置、目标总数和置信度信息
- 支持自定义检测参数（置信度、IOU阈值）
- 提供模型选择功能和检测结果保存功能
- 支持批量检测并直接查看所有结果
- 包含完整训练好的模型及训练结果

## 适用场景
- 农业科研与病害研究
- 水稻种植病害监测
- 农业技术推广与教学
- 农田病虫害普查

## 技术特点
- 采用Pycharm+Anaconda或VSCode+Anaconda开发环境
- 基于python3.9、opencv-python、PyQt5、ultralytics、torch1.9等技术栈
- 提供完整UI界面源文件和图标资源
- 包含详细的项目运行教程和文档资料
